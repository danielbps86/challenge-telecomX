<h1>Challenge Telecom X</h1>

Segundo challenge de Data Science del programa ONE de alura LATAM. Mediante este challenge se analiza base de datos de la empresa Telecom X para determinar que está ocasionando la fuga de clientes de la compañía.

¿Cómo ejecutar el código? El código se encuentra en el archivo TelecomX_LATAM.ipynb. Debe descargar el archivo y abrirlo en un entorno que soporte cuadernos Jupyter, se recomienda el uso de de la herramienta en la nube Google Colab. El coódigo se encuentra organizado en bloques como el siguiente, en los cuales, se analizan distintos aspectos que pudiesen estar ocasionando la fuga de clientes de la empresa Telecom X.

![imagen](https://github.com/user-attachments/assets/0c0044cf-631f-411a-992a-9d31cdeee8a6)


En las distintas secciones, observará diferentes gráficos, los cuales, muestran de manera visual los resultados de cada análisis.

![imagen](https://github.com/user-attachments/assets/e6166e8b-cfb2-44a7-aee2-845f4b678ffb)


Al final del archivo, encontrará un informe con el análisis de los datos presentados, le cual presenta las conclusiones acerca de las causas de la deserción de los clientes, y las recomendaciones para atraer y fidelizar nuevos clientes.
